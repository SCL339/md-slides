# md-slides

**Markdown to HTML Presentations**

Made with ❤️ by SCL339

---

## Why md-slides?

- **Zero config:** Write Markdown, get HTML
- **Single file:** Self-contained, no internet needed
- **Dark theme:** Modern and easy on the eyes
- **Keyboard nav:** Arrow keys, Space, Home, End

---

## Features

- Slide transitions (fade + slide-up)
- Progress bar
- Slide counter
- Touch swipe support
- Fullscreen (press `f`)
- Watch mode (`--watch`)

---

## How It Works

```markdown
# Slide 1
Content here

---

# Slide 2
More content
```

Separate slides with `---`.

---

## Installation

```bash
npx md-slides presentation.md
```

No install needed — just npx!

---

## Customization

- `-o, --output` — Custom output path
- `-t, --template` — Custom HTML template
- `-w, --watch` — Watch for changes

---

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| → / ↓ | Next slide |
| ← / ↑ | Previous slide |
| Space | Next slide |
| f | Fullscreen |
